## HTML

- `<br>` : 라인 이동

- `<p>` : 단락이동

- `<a href="페이지 url" target="창 뜨는 옵션"> <img src="image.jpg"> or 글자 </a>` : 페이지 링크

- `<img src="image.jpg title="풍선 도움말">` : 이미지 삽입, 형식 (png, gif, jpg)

- `<div>` : 페이지 레이아웃

  ​		 	  **block** tag : 자동으로 라인 이동이 발생하는 태그

-  `<span>` : 페이지 레이아웃

  ​				**inline** tag : 라인 이동이 발생하지 않는 태그

- `<hr>` : 수평선
